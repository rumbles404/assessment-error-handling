# PaymentServiceAssessment

This project is a Senior Java SDET code review assessment focused on error handling using TestNG and Gradle.

## Structure

- `PaymentService.java`: Contains intentional error handling issues.
- `PaymentGateway.java`: Interface for simulating payment processing.
- `PaymentServiceTest.java`: TestNG-based unit tests.

## Instructions

1. Review the `PaymentService` class and identify at least 5 issues.
2. Suggest improvements or refactor the code.
3. Run tests using:

```bash
./gradlew test
```

On Windows

gradlew.bat test
